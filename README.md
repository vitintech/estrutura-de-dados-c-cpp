# 📚 Estrutura de Dados em C/C++

Este repositório documenta meu processo de aprendizado em **Estrutura de Dados utilizando C e C++**, desenvolvido a partir da disciplina da faculdade e complementado com estudos e cursos por fora.

O foco não é apenas resolver exercícios, mas **entender, implementar, documentar e evoluir** cada estrutura e algoritmo.

---

## 🎯 Objetivo

- Consolidar fundamentos de Estrutura de Dados
- Aplicar conceitos teóricos na prática
- Desenvolver código limpo, organizado e comentado
- Demonstrar evolução contínua no aprendizado em C/C++

---

## 🧠 Conteúdos Abordados

- Arrays e Matrizes
- Listas Ligadas (Simples, Duplas, Circulares)
- Pilhas (Array e Lista)
- Filas (Simples e Circulares)
- Árvores (Binárias, BST)
- Grafos (representações e algoritmos básicos)
- Algoritmos de Ordenação

---

## 🛠️ Tecnologias Utilizadas

- Linguagem C
- Linguagem C++
- GCC / G++
- Estrutura modular de código
- Comentários explicativos e documentação

---

## 📂 Organização do Repositório

Cada pasta representa uma estrutura ou conceito estudado.  
Dentro delas, você encontrará:

- Implementações práticas
- Comentários explicativos
- Observações sobre complexidade
- README local explicando o conceito

---

Este repositório está em **atualização contínua**, acompanhando minha evolução na disciplina e estudos externos.
